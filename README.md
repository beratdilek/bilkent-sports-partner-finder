# bilkent-sports-partner-finder

A Moodle-ready, single-file web feature that helps university students find sports partners and organize activities.

## Features
- Sport-based tabs (add new sports dynamically)
- Create and join activities with participant limits
- Date & time validation (prevents past events)
- Search, filter (available only), and sorting (nearest / newest)
- Data persistence using localStorage with graceful fallback
- Accessible UI (ARIA roles, keyboard-friendly modal)

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript (DOM API)
- localStorage
